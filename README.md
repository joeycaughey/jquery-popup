jquery-popup
============

Multipurpose jQuery Popup that auto positions and auto recognizes form and POST action behaviours.

----------------------------------------------------
USAGE AND INITIALIZATION
----------------------------------------------------

Load a URL.
```
Popup.init({
  width: 300,
  height : 400,
  variables : {},
  url : 'popup.html',
  nopadding: true
});
```

Load HTML content.
```
Popup.init({
  width: 300,
  height : 400,
  content : '<b>HTML TEXT</b>',
  nopadding: true
});
```
